### University of Alabama at Birmingham (UAB)

This repository contains my lab assignments, homework, and practice code for **CS 332 – Systems Programming**, taken during **Summer 2026**. All work is written in **C** and focuses on low‑level programming, Unix systems, memory management, and algorithm implementation.

---

## 📘 Course Overview

CS 332 covers foundational systems‑level programming topics, including:

- C programming fundamentals  
- Pointers and pointer arithmetic  
- Dynamic memory allocation (`malloc`, `free`)  
- Arrays and strings in C  
- Unix architecture and system internals  
- Compiling and debugging with `gcc` and `gdb`  
- Implementing algorithms (e.g., insertion sort)  
- Working directly with memory and low‑level data structures  

The course emphasizes writing correct, efficient, and memory‑safe C programs.

## 🎯 Course Objectives

Upon completion, students will be able to:

- Write, debug, and execute C programs in a Unix environment  
- Understand Unix architecture and system behavior  
- Use dynamic memory allocation effectively  
- Manipulate strings and arrays at a low level  
- Implement algorithms using C and pointers  
- Work with system‑level tools and compilers  

ABET outcomes include designing computing solutions and functioning effectively in team environments.
